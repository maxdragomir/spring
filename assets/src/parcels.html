How to install thermal printer driver

Click here to download printer driver and save it on your computer.

Run downloaded file eplprinter.exe (double click on it). On Windows 7 and higher you may need Administrator priveledges on first run to allow install driver in the computer's registry.

In opened window, select thermal printer from the list of installed printers and click save. NOTE: printer can be a network printer (connected to another computer and shared).

Browser will ask you permission to run this software (eplprinter) at first label print.
